# Skeleton Hunt
Kill all the Skeleton present in scene to complete the Level.
Created using Flutter and Flame Engine.
Created for Midyear 2022 Flame Game Jam

## Controls
A - Move Left
D - Move Right
W - Attack
Space - Jump
R - Pause/Resume
